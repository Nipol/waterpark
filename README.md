Waterpark
=========

베리즈가 성능도 안 나오고, 보안성도 구리고 사용성도 구려서 새롭게 만들어 보려고 합니다.

Start Dev!
------------
```
  $ git clone git@github.com:Nipol/waterpark.git
  $ cd waterpark/
  $ vagrant up
  $ vagrant ssh
```
